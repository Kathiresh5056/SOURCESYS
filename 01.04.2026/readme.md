Project: Sentiment Analysis using Hugging Face

This project implements a **Sentiment Analysis system** using a pre-trained transformer model from Hugging Face.  
It classifies user input text as **Positive** or **Negative**.


- Uses pre-trained NLP model (no training required)
- Real-time sentiment prediction
- Simple command-line interface
- Modular code structure (model + utils)



 Model Used

- Model: distilbert-base-uncased-finetuned-sst-2-english
- Library: Hugging Face Transformers
- Task: Sentiment Analysis (Binary Classification)
